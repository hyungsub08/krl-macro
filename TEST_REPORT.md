## 6. 수정 이력

### 취소 API 버그 수정 (이번 테스트 중 발견 및 수정)

| 항목 | 내용 |
|---|---|
| 파일 | `korail2/korail2.py` (`cancel` 함수) |
| 증상 | `'int' object is not subscriptable` |
| 원인 | `self._session.get(url, data=data)` — GET 요청에 `data`를 전달하면 request body로 전송되어 서버가 400 Bad Request 반환. 400 응답을 `json.loads()`하면 정수가 반환되어 후속 코드에서 subscript 에러 발생 |
| 수정 | `data=data` → `params=data` (쿼리스트링으로 전송) |
| 검증 | 수정 후 예매+취소 전체 PASS 확인 |
